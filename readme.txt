WPDaddy header builder was developed for Elementor page builder.

== Description ==

WPDaddy header builder is a visual front-end drag & drop builder for WordPress Elementor plugin. You can easily build any header on your website developed with Elementor page builder. It's very easy and intuitive WordPress header builder.

It works with any WordPress theme, no limits.

**WPDaddy Header Builder Advantages:**

* Live preview in Elementor
* Unlimited headers
* Custom widgets for header builder
* Advanced settings for custom widgets
* Works with any Elementor widget
* Select any area on site to load header
* Unlimited levels of header rows
* Fully responsive. Desktop, tablet or mobile
* Sticky options for each header row
* Easy import/export headers
* Advanced header settings panel
* Extensive documentation and support
* New features coming soon

== Installation ==

After downloading the ZIP file,

1. Log in to WordPress dashboard panel.
2. Go to Plugins Add > New > Upload.
3. Click Choose file (Browse) and select the downloaded zip file of the WPDaddy header builder.
4. Click Install Now button.
5. Click Activate Plugin button for activating the WPDaddy header builder plugin.
If the installation does not succeed, please contact us.
